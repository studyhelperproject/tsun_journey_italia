# アセット要件定義書：トップ画面 〜 ステージ1クリアまで

本作「おぱのイタリアン脱出（仮）」のプロトタイプ構築に必要なアセットのリストおよび要件を定義します。

## 1. デザインガイドライン（共通要件）

全ての視覚アセットは、以下の「コミック・ラインアート」スタイルに従って制作してください。

- **外形線**: 2pxの太い黒の実線（`#222222`）。
- **影**: ぼかしのない、硬いドロップシャドウ（4pxオフセット、`#222222`）。
- **カラーパレット**:
  - メイン：イタリアングリーン (`#008C45`)
  - ベース：オフホワイト (`#F4F5F0`)
  - アクセント：イタリアンレッド (`#CD212A`)
  - サブアクセント：パスタゴールド (`#FFD700`)
- **質感**: グラデーションを避け、フラットな塗り（セル画風）を基本とする。

| :--- | :--- | :--- |

## 3. サウンドアセット・リスト

### 3.1 BGM (MP3/WAV)
- **タイトル画面BGM**: 明るく、アコーディオンやマンドリンを使ったイタリアンな楽曲。
- **メインゲームBGM**: リラックスしたカフェの雰囲気。少しシュールな間があるとなお良い。

### 3.2 SE (効果音)
- **決定/タップ音**: ポンッ、という小気味よい音。
- **アイテム入手音**: キラキラした短い音。
- **アイテム使用（正解）**: ピンポン！という正解音。
- **アイテム使用（不正解）**: ブブー！、または軽い打撃音。
- **ステージクリア**: ファンファーレ（イタリア風の陽気な旋律）。
- **ゲームオーバー**: 悲劇的なオペラの一節、またはガーンという音。
- **だでの叫び**: 「Mamma mia!」や「Nooo!」といったボイス。

## 4. テクニカルスペック

- **画像フォーマット**: PNG（透過必須）、またはUI要素はSVG。
- **解像度**:
  - 背景：2436 x 1125 px (iPhone X以降の全画面対応を想定)
  - キャラクター：最低 512 x 512 px（拡大してもぼやけないサイズ）
- **オーディオ**:
  - BGM：ループ再生可能なファイル形式。
  - SE：遅延の少ないWAV形式推奨。
