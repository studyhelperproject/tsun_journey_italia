# 画面設計計画書

## 1. デザインシステム (Design System)

本作の「ゆる脱出ゲーム」というコンセプトと、イタリアのテーマを反映したデザインガイドラインです。

### カラーパレット
- **メインカラー (イタリアングリーン)**: `#008C45` - ヘッダーや成功演出のアクセントに使用。
- **ベースカラー (オフホワイト)**: `#F4F5F0` - 背景色。ピザ生地や石造りの建物をイメージ。
- **アクセントカラー (イタリアンレッド)**: `#CD212A` - ボタンや失敗演出、警告に使用。
- **サブアクセント (パスタゴールド)**: `#FFD700` - アイテムスロットの枠や重要アイテムに使用。

### タイポグラフィ
- **フォントスタイル**: 丸みを帯びたサンセリフ体（Rounded Sans-serif）。
- **用途**:
  - タイトル・見出し: 太字で親しみやすい印象。
  - 本文・メッセージ: 読みやすさを重視し、適度な行間を確保。

### 共通コンポーネント
- **ボタン**:
  - サイズ: 最小 44x44 dp（タッチ操作のしやすさを考慮）。
  - スタイル: 角丸（Radius: 10dp）、軽いドロップシャドウで押しやすさを強調。
- **アイテムスロット**:
  - サイズ: 60x60 dp。
  - 背景: 薄いグレーまたはゴールドの枠線。

## 2. 画面一覧 (Screen Inventory)

### 既存計画に基づく画面
- **メインゲーム画面 (Main Game Screen)**: ギミック操作やアイテム使用を行う中心となる画面。
- **成功演出画面 (Success Overlay)**: ステージクリア時に表示される。おぱの喜ぶ姿を表示。
- **失敗演出画面 (Failure Overlay)**: 禁忌を犯した際に表示される。だでの絶望や怒りを表示。

### 追加で必要な画面（アプリフロー完成のため）
- **タイトル画面 (Title Screen)**: アプリ起動時の画面。スタートボタン、ステージ選択への遷移。
- **ステージ選択画面 (Stage Select Screen)**: 全20ステージのリスト。開放状況を表示。
- **メニュー画面 (Menu Overlay)**: 設定（音量など）やタイトルへ戻る機能。
- **ヒント画面 (Hint Overlay)**: つんからのヒントを表示する画面。

---

## 3. コンポーネント配置とレイアウト (Component Layouts)

スマートフォンの標準的な解像度（375x812 dp程度）を基準に設計します。

### メインゲーム画面の構成
- **ヘッダー (Header)**:
  - **高さ**: 60 dp
  - **配置**:
    - 左側: ステージ番号表示（例：「Stage 01」）。
    - 右側: ヒントボタン（44x44 dp）、メニューボタン（44x44 dp）。
- **メインゲームエリア (Main Area)**:
  - **高さ**: フレキシブル（ヘッダーとフッターの間すべて）。
  - **内容**: キャラクター（おぱ、つん、だで）、インタラクティブなオブジェクト。
  - **機能**: タップ検知、アイテムのドラッグ＆ドロップ受け入れ。
- **フッター (Footer)**:
  - **高さ**: 100 dp
  - **配置**:
    - 中央にアイテムスロットを最大5枠横並びに配置。
    - 各スロットは 60x60 dp。スロット間の余白は 8 dp。

### オーバーレイ画面（成功・失敗・メニュー等）
- **全体**: 背景を半透明（黒 50%）にしてメイン画面の上に重ねる。
- **中央パネル**:
  - キャラクターのイラスト、メッセージ、アクションボタン（「次へ」「リトライ」「ステージ選択へ」など）を配置。
