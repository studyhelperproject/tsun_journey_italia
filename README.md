# おぱのイタリアン脱出（仮）

## プロジェクト概要
本作は、イタリアの食文化やマナーを題材にしたカジュアル脱出パズルゲームです。
主人公「おぱ」がイタリアで禁忌を犯さないよう、プレイヤーがギミックを操作してクリアを目指します。

## 主なキャラクター
- **おぱ**: 主人公。ついつい禁忌を犯してしまう天然キャラ。
- **つん**: 味方。おぱを助ける親友。
- **だで**: 敵/お邪魔キャラ。イタリアの伝統を守る厳しい存在。
